Web Scraping

Descripción general

Este proyecto utiliza Python y la biblioteca Selenium para automatizar el proceso de descarga de reportes desde la plataforma Salesys. El objetivo es simplificar y agilizar la obtención de datos relevantes, minimizando la intervención manual y reduciendo el tiempo necesario para realizar esta tarea.

Funcionalidad del proyecto

  1. Inicio de Sesión en Salesys:
    • Se abre el navegador y se navega a la página de inicio de Salesys.
    • Se ingresan las credenciales necesarias (usuario y contraseña) para iniciar sesión.
  2. Acceso al reporte:
    • Una vez dentro, el script navega a la sección de reportes.
    • Se selecciona el reporte que se desea descargar, configurando automáticamente la fecha al día anterior.
  3. Esperar carga de la tabla:
    • Se implementa una espera explícita para asegurarse de que la tabla con los datos esté completamente cargada antes de proceder a la descarga.
  4. Descarga del reporte:
    • Una vez que la tabla está lista, el script inicia la descarga del reporte en el formato deseado.
  5. Cierre de sesiones y navegador:
    • Después de completar la descarga, el script cierra la sesión en Salesys y cierra el navegador, asegurando que no queden ventanas abiertas.

Tecnologías utilizadas

  • Python: Lenguaje de programación principal utilizado para desarrollar el script.
  • Selenium: Biblioteca para la automatización de navegadores web.
  • WebDriver: Controlador de Selenium utilizado para interactuar con el navegador (Chrome, Firefox, etc.).

Resultados y conclusiones

Este proyecto permite la descarga automatizada de reportes de Salesys, lo que ahorra tiempo y esfuerzo en comparación con la ejecución manual del proceso. La automatización proporciona un enfoque más eficiente, permitiendo a los usuarios centrarse en el análisis de datos en lugar de en la recopilación.
